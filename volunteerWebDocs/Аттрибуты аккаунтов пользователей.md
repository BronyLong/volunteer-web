
==--------------Профили--------------

***Обязательные поля для каждого пользователя 

| Название поля | Тип данных   | Описание                                                 |
| ------------- | ------------ | -------------------------------------------------------- |
| id            | int или uuid | Уникальный идентификатор пользователя                    |
| email         | varchar(255) | Электронная почта                                        |
| password_hash | varchar      | Хеш пароля                                               |
| role          | под вопросом | Роль пользователя (волонтер, координатор, администратор) |
| created_at    | timestamp    | Время создания профиля                                   |
| updated_at    | timestamp    | Время изменения профиля (если проводились изменения)     |
| is_active     | BOOLEAN      | Активен ли профиль (может быть заблокирован)             |
| last_login    | TIMESTAMP    | Время последнего входа                                   |

***Профиль пользователей

| Название поля | Тип данных   | Описание                                                 | Обязательный |
| ------------- | ------------ | -------------------------------------------------------- | ------------ |
| user_id       | FK           | Ссылка на универсальные поля, данные связаны с аккаунтом | ✅            |
| first_name    | varchar(100) | Имя                                                      | ✅            |
| last_name     | varchar(100) | Фамилия                                                  | ✅            |
| middle_name   | varchar(100) | Отчество                                                 | ❌            |
| phone         | varchar(20)  | Телефон                                                  | ❌            |
| city          | varchar(100) | Город                                                    | ❌            |
| avatar_url    | TEXT         | Ссылка на фото профиля                                   | ❌            |
| bio           | TEXT         | Описание / Информация о себе                             | ❌            |
| date_of_birth | DATE         | Дата рождения                                            | ✅            |

***Профиль координатора


| Название поля | Тип данных   | Описание                                                                          | Обязательный |
| ------------- | ------------ | --------------------------------------------------------------------------------- | ------------ |
| user_id       | FK           | Ссылка на универсальные поля, данные связаны с аккаунтом                          | ✅            |
| first_name    | varchar(100) | Имя                                                                               | ✅            |
| last_name     | varchar(100) | Фамилия                                                                           | ✅            |
| middle_name   | varchar(100) | Отчество                                                                          | ❌            |
| phone         | varchar(20)  | Телефон                                                                           | ✅            |
| avatar_url    | TEXT         | Ссылка на фото профиля                                                            | ❌            |
| bio           | TEXT         | Описание / Информация о себе                                                      | ❌            |
| position      | VARCHAR(100) | Название должности (волонтер группы ... или администратор волонтерского движения) | ✅            |
| department    | VARCHAR(100) | Каким подразделением руководит                                                    | ✅            |

***Профиль администратора (опционально)

| Название поля | Тип данных   | Описание                                                 | Обязательный |
| ------------- | ------------ | -------------------------------------------------------- | ------------ |
| user_id       | FK           | Ссылка на универсальные поля, данные связаны с аккаунтом | ✅            |
| first_name    | varchar(100) | Имя                                                      | ✅            |
| last_name     | varchar(100) | Фамилия                                                  | ✅            |
| phone         | varchar(20)  | Телефон                                                  | ✅            |

==--------------Дополнительные таблицы--------------

***Дополнительная таблица для категорий для волонтеров

| Название поля | Тип данных | Описание                    |
| ------------- | ---------- | --------------------------- |
| id            | SERIAL     | Primary key                 |
| user_id       | FK         | ID волонтера                |
| category_id   | FK         | ID категории                |
| created_at    | TIMESTAMP  | Когда создана (опционально) |

***Таблица ролей (под вопросом)

| Название поля | Тип данных  | Описание                                |
| ------------- | ----------- | --------------------------------------- |
| id            | SERIAL      | Primary key                             |
| name          | varchar(12) | Имя роли (админ, координатор, волонтер) |

***Теги (категории)

| Название поля | Тип данных | Описание                                                      |
| ------------- | ---------- | ------------------------------------------------------------- |
| category_id   | SERIAL     | ID категории                                                  |
| name          | VARCHAR    | Название категории (например "животные" или "помощь пожилым") |

***Учет часов

| Название поля | Тип данных | Описание                         |
| ------------- | ---------- | -------------------------------- |
| id            | SERIAL     | ID для учета часов               |
| user_id       | FK         | Кто участник                     |
| event_id      | FK         | В каком мероприятии              |
| hours_worked  | INT        | Сколько часов отработал          |
| confirmed_by  | FK         | Кто подтвердил (ID координатора) |
| confirmed_at  | TIMESTAMP  | Во сколько подтвердил            |

***Регистрация на мероприятия

| Название поля | Тип данных  | Описание                                            |
| ------------- | ----------- | --------------------------------------------------- |
| id            | SERIAL      | ID для учета часов                                  |
| user_id       | FK          | Кто участник                                        |
| event_id      | FK          | В каком мероприятии                                 |
| status        | VARCHAR(20) | Статус заявки на участие(ожидает, принят, отклонен) |
| created_at    | TIMESTAMP   | Время создания заявки                               |
| updated_at    | TIMESTAMP   | Время обновления заявки                             |
